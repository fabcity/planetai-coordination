PLANETAI is a federated AI causal-inference instrument for distributed production at bioregional scale. Four pilots — Barcelona, Boston, Santiago, Bali — coupled to Earth foundation models, community sensor networks, and the global fab-lab infrastructure.

This site is the coordination layer for the program: what we have built, the decisions that shaped it, and the work-tracks open for partner contribution.

Contribute

Most coordination work happens by editing files in this repo. Three paths in, depending on what you want to do:

  • Propose a decision — open a structured form that becomes a GitHub issue. No YAML knowledge required; Tomas or Lucas converts it to a YAML file when triaging.
  • Ack or push back on a review — open the response form. Same idea, scoped to review windows.
  • Edit a YAML directly — on every decision page, the Edit this decision on GitHub link opens the source file in GitHub's web editor. Save = pull request. No terminal involved.
  • Email or Slack Tomas / Lucas — if none of the above fit. We'll log the YAML for you. Honest fallback.

A free GitHub account is required for the first three. Setup takes sixty seconds at github.com/signup.
